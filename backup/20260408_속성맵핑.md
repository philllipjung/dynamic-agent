# 속성 매핑 완전 가이드

**작성일:** 2026-04-08
**목적:** Java Agent에서 Arthas Watch로 파라미터를 추출하고, ParameterMappingService로 매핑하여 ByteBuddy Advice가 Span 속성으로 저장하는 전체 과정 상세 설명

---

## 1. 개요

이 문서는 **Java Agent 시스템에서 메서드의 파라미터를 Arthas Watch로 확인하고, 사용자가 지정한 이름으로 매핑하여 OpenTelemetry Span의 속성으로 자동 저장하는 과정**을 실제 코드와 데이터 흐름과 함께 설명합니다.

### 왜 속성 매핑이 필요한가요?

```
문제 상황:
- Arthas Watch 결과: "params[0]=user123, params[1]=ORD-001"
- 문제: 이게 userId인지 orderId인지 알 수 없음!
- 개발자는 코드를 보고 "아, params[0]이 userId구나"

해결:
- 사용자가 한 번만 매핑: "params[0] → userId"
- 그 다음부터는 자동으로: span.setAttribute("arthas.attribute.userId", "user123")
- OpenSearch에서 "userId로 검색" 가능!
```

public static JSONObject watchWithParameters(String className, String methodName, int limit) {
    // 1. 사용 가능한 PID 찾기
    String pid = getFirstAvailablePid();  // "15428"

    // 2. Watch 명령 생성
    String watchCmd = String.format("watch %s %s params -n %d -x 1",
        className, methodName, limit);
    // → "watch com.test.service.test1.controller.Test1Controller test1 params -n 1 -x 1"

    System.out.println("[ArthasManager] Executing watch: " + watchCmd);

    // 3. Arthas HTTP API로 명령 실행
    String result = executeTunnelApiCommand(pid, watchCmd);

    // 4. JSON 응답 파싱
    JSONObject response = new JSONObject(result);

    // 5. 상태 확인
    String state = response.optString("state", "UNKNOWN");
    if (!"SUCCEEDED".equals(state)) {
        return new JSONObject().put("error", "Command execution failed: " + state);
    }

    // 6. 파라미터 추출
    JSONArray parameters = new JSONArray();
    if (response.has("body")) {
        JSONObject body = response.getJSONObject("body");
        if (body.has("results")) {
            JSONArray results = body.getJSONArray("results");

            for (int i = 0; i < results.length(); i++) {
                JSONObject resultItem = results.getJSONObject(i);

                if ("watch".equals(resultItem.optString("type"))) {
                    String value = resultItem.optString("value", "");
                    System.out.println("[ArthasManager] Watch value: " + value);

                    // 7. Arthas 출력 파싱
                    // "@Object[@String[0006]]" → [{"index":0, "type":"String", "value":"0006"}]
                    parameters = parseWatchValue(value);
                }
            }
        }
    }

    // 8. 결과 생성
    JSONObject resultJson = new JSONObject();
    resultJson.put("success", true);
    resultJson.put("className", className);
    resultJson.put("methodName", methodName);
    resultJson.put("parameters", parameters);
    resultJson.put("parameterCount", parameters.length());

    System.out.println("[ArthasManager] Extracted " + parameters.length() + " parameters");

    return resultJson;
}

// ArthasManager.java line 300-362
private static JSONArray parseWatchValue(String value) {
    JSONArray parameters = new JSONArray();

    // Arthas 출력 예시: "@Object[@String[0006]]"

    // 1. 따옴표 제거
    value = value.replace("\\\"", "\"");
    value = value.replace("\\\\", "\\");

    // 2. 정규식으로 파라미터 추출
    Pattern pattern = Pattern.compile("params\\[(\\d+)\\]\\s*=\\s*\"?([^\"]+)\"?");
    Matcher matcher = pattern.matcher(value);

    Set<Integer> foundIndices = new HashSet<>();

    while (matcher.find()) {
        int index = Integer.parseInt(matcher.group(1));  // "0" → 0
        String paramName = matcher.group(2).trim();     // "0006"

        JSONObject paramInfo = new JSONObject();
        paramInfo.put("index", index);
        paramInfo.put("name", paramName);        // "0006"
        paramInfo.put("type", "String");
        paramInfo.put("value", paramName);        // "0006"
        paramInfo.put("extracted", true);

        parameters.put(paramInfo);
        foundIndices.add(index);
    }

    // 3. params[N] 패턴이 없으면 대안 파싱
    if (foundIndices.isEmpty()) {
        // Object[] 배열 형식 파싱: "@Object[@String[0006]]"
        String[] parts = value.split("[,\\[\\]]");
        int idx = 0;
        for (String part : parts) {
            part = part.trim();
            if (!part.isEmpty() && !part.contains("params")) {
                // @String[0006] → "0006"
                if (part.contains("@")) {
                    int bracketStart = part.lastIndexOf('[');
                    int bracketEnd = part.indexOf(']', bracketStart);
                    if (bracketStart > 0 && bracketEnd > bracketStart) {
                        String actualValue = part.substring(bracketStart + 1, bracketEnd);
                        paramName = actualValue;
                    }
                }

                JSONObject paramInfo = new JSONObject();
                paramInfo.put("index", idx++);
                paramInfo.put("name", paramName);
                paramInfo.put("type", "Object");
                paramInfo.put("value", paramName);
                parameters.put(paramInfo);
            }
        }
    }

    return parameters;
}
```

// ArthasController.java line 360-404
@PostMapping("/mapParameters")
public Map<String, Object> mapParameters(@RequestBody MapParameterRequest request) {
    // 1. 요청 데이터 추출
    String className = request.getClassName();    // "com.test.service.test1.controller.Test1Controller"
    String methodName = request.getMethodName();  // "test1"
    Map<String, String> mapping = request.getMapping();  // {"0": "userId"}

    // 2. Map<String, String> → Map<Integer, String> 변환
    Map<Integer, String> paramMapping = new LinkedHashMap<>();

    if (request.getMapping() != null) {
        for (Map.Entry<String, String> entry : request.getMapping().entrySet()) {
            try {
                // "0" → 0
                int index = Integer.parseInt(entry.getKey());
                String paramName = entry.getValue();  // "userId"
                paramMapping.put(index, paramName);
            } catch (NumberFormatException e) {
                return Map.of("success", false, "error", "Invalid parameter index: " + entry.getKey());
            }
        }
    }

    // → paramMapping = {0: "userId"}

    // 3. ParameterMappingService로 Redis에 저장
    parameterMappingService.saveMapping(className, methodName, paramMapping);

    // 4. SpanAdvice 캐시에도 저장 (런타임에서 바로 사용)
    SpanAdvice.setParameterMapping(className, methodName, paramMapping);

    return Map.of(
        "success", true,
        "message", "매핑 저장 완료 (Redis + Advice Cache)",
        "className", className,
        "methodName", methodName,
        "mapping", paramMapping,
        "storage", "Redis + SpanAdvice"
    );
}
```

```java
// agent-server/.../ParameterMappingService.java (Redis 연동 버전)
@Service
public class ParameterMappingService {

    @Autowired
    private RedisTemplate<String, String> redisTemplate;

    private static final ObjectMapper objectMapper = new ObjectMapper();
    private static final String KEY_PREFIX = "paramMapping:";

    public void saveMapping(String className, String methodName, Map<Integer, String> mapping) {
        // 1. Redis Key 생성
        String key = KEY_PREFIX + className + ":" + methodName;
        // → "paramMapping:com.test.service.test1.controller.Test1Controller:test1"

        // 2. Map<Integer, String> → Map<String, String> 변환 (JSON 직렬화용)
        Map<String, String> stringMap = new HashMap<>();
        for (Map.Entry<Integer, String> entry : mapping.entrySet()) {
            stringMap.put(entry.getKey().toString(), entry.getValue());
        }
        // → {"0": "userId"}

        // 3. JSON으로 변환
        String jsonValue = objectMapper.writeValueAsString(stringMap);
        // → "{\"0\":\"userId\"}"

        // 4. Redis에 저장
        redisTemplate.opsForValue().set(key, jsonValue);

        System.out.println("[ParameterMappingService] Saved mapping: " + key + " = " + jsonValue);
    }
}

// bytebuddy-agent/.../ByteBuddyAgent.java line 605-663
public static String createSpanAttributeAdvice(String pid, String className, String methodName,
                                               Map<Integer, String> paramMapping) {
    try {
        // 1. 매핑을 SpanAdvice에 등록
        SpanAdvice.setParameterMapping(className, methodName, paramMapping);

        // 2. 타겟 클래스 로드
        Class<?> targetClass = Class.forName(className);
        ClassLoader targetClassLoader = targetClass.getClassLoader();

        // 3. Helper 클래스 주입
        injectHelper(targetClassLoader);
        // → SpanHelper, SpanAttributeHelper, ThreadNameHelper 등을
        //    targetClassLoader에 로드하여 Advice에서 사용 가능하게 함

        // 4. SpanAdvice 로드
        Class<?> adviceClass = targetClassLoader.loadClass(
            "com.javaagent.bytebuddy.advices.SpanAdvice"
        );

        // 5. ByteBuddy로 바이트코드 생성
        byte[] transformedBytes = new ByteBuddy()
            .redefine(targetClass)
            .visit(Advice.to(adviceClass)
                .on(ElementMatchers.named(methodName)
                        .and(ElementMatchers.isMethod())
                        .and(ElementMatchers.not(ElementMatchers.isStatic()))))
            .make()
            .getBytes();

        // 6. 재변환 적용
        java.lang.instrument.ClassFileTransformer transformer = new java.lang.instrument.ClassFileTransformer() {
            @Override
            public byte[] transform(ClassLoader loader, String className,
                                  java.security.ProtectionDomain protectionDomain, byte[] classfileBuffer) {
                if (className.replace('/', '.').equals(targetClass.getName())) {
                    System.out.println("[ByteBuddy] *** RETRANSFORMING: " + className + " ***");
                    return transformedBytes;
                }
                return null;
            }
        };

        instrumentation.addTransformer(transformer, true);
        instrumentation.retransformClasses(targetClass);
        instrumentation.removeTransformer(transformer);

        System.out.println("[ByteBuddy] *** SPAN ATTRIBUTE ADVICE APPLIED TO " + className + "." + methodName + " ***");
        return "SUCCESS: Created span attribute advice for " + className + "." + methodName;

    } catch (Exception e) {
        e.printStackTrace();
        return "ERROR: Failed to create span attribute advice - " + e.getMessage();
    }
}

        // → "test1"
        String key = className + "." + methodName;
        // → "com.test.service.test1.controller.Test1Controller.test1"

        // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
        // 2. userId 추출 (매핑에서)
        // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
        Map<Integer, String> paramMapping = parameterMappings.get(key);
        if (paramMapping != null && allArguments != null) {
            for (Map.Entry<Integer, String> entry : paramMapping.entrySet()) {
                if ("userId".equals(entry.getValue())) {
                    Object value = allArguments[entry.getKey()];
                    if (value != null) {
                        userId = value.toString();
                    }
                }
            }
        }
    }

    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
    // 3. Jaeger Link 생성 (선택적)
    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
    List<SpanContext> linkedContexts = Collections.emptyList();
    if (ENABLE_JAEGER_LINK && userId != null && className != null) {
        try {
            linkedContexts = findLinkedContexts(className, methodName, userId);
        } catch (Exception e) {
            System.err.println("[SpanAdvice] Error finding links: " + e.getMessage());
        }
    }

    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
    // 4. Span 생성 (Links 포함)
    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
    SpanHelper spanHelper = SpanHelper.createSpanWithLinks(method, linkedContexts);

    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
    // 5. 스레드명 변경 (TraceId + SpanId)
    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
    System.err.println(">>> [ThreadName] Check: spanHelper=" + (spanHelper != null && spanHelper.isValid()) + ", isTarget=" + ThreadNameHelper.isTargetThread());
    if (spanHelper != null && spanHelper.isValid() && ThreadNameHelper.isTargetThread()) {
        System.err.println(">>> [ThreadName] Renaming thread...");
        ThreadNameHelper.renameThread(spanHelper.getSpan());
        renamedThread = true;
        spanHelper.setOriginalThreadName(originalThreadName);
    }

    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
    // 6. 파라미터 속성 추가
    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ //
    if (spanHelper != null && spanHelper.isValid() && target != null) {
        try {
            String key = className + "." + methodName;
            System.err.println(">>> DEBUG: Looking up key: " + key);
            System.err.println(">>> DEBUG: userId=" + userId);

            Map<Integer, String> paramMapping = parameterMappings.get(key);
            System.err.println(">>> DEBUG: paramMapping=" + paramMapping);

            if (paramMapping != null && !paramMapping.isEmpty() && allArguments != null) {
                SpanAttributeHelper attrHelper = new SpanAttributeHelper(spanHelper.getSpan());

                for (Map.Entry<Integer, String> entry : paramMapping.entrySet()) {
                    int paramIndex = entry.getKey();      // 0
                    String paramName = entry.getValue();    // "userId"

                    if (paramIndex < allArguments.length) {
                        Object paramValue = allArguments[paramIndex];  // "customUserId"

                        // 속성 추가 (자동으로 "arthas.attribute." prefix 추가)
                        attrHelper.setParameterAttribute(paramName, paramValue);
                        // → span.setAttribute("arthas.attribute.userId", "customUserId")

                        System.err.println(">>> Set attribute: " + paramName + " = " + paramValue);
                    }
                }
            }
        } catch (Exception e) {
            System.err.println(">>> Failed to set attributes: " + e.getMessage());
            e.printStackTrace();
        }
    }

    // 7. SpanHelper를 반환 (onMethodExit에서 사용)
    return spanHelper;
}
```

    private static final String ATTRIBUTE_PREFIX = "arthas.attribute.";
    private final Span span;

    // 파라미터 속성 추가
    public SpanAttributeHelper setParameterAttribute(String paramName, Object paramValue) {
        if (paramName == null || paramName.isEmpty()) {
            return this;
        }

        String value = convertToString(paramValue);  // "customUserId"
        setAttribute(paramName, value);              // "userId", "customUserId"
        return this;
    }

    // 속성 추가 (자동으로 prefix 추가)

            String fullKey = ATTRIBUTE_PREFIX + key;  // "arthas.attribute.userId"
            span.setAttribute(fullKey, value);         // OpenTelemetry API
        }
        return this;
    }
}

public static SpanHelper onMethodEnter(...) {

**핵심 포인트:**
1. **Arthas Watch**는 파라미터 **인덱스와 값**만 알 수 있음
2. **사용자 매핑**으로 인덱스에 **이름**을 부여
3. **매핑은 Redis + Memory** 두 곳에 저장 (안정성 + 성능)
4. **런타임**에는 SpanAdvice가 매핑을 조회하여 자동으로 Span 속성 추가
5. 모든 속성은 `arthas.attribute.` prefix로 저장되어 OpenSearch에서 쉽게 검색 가능!
