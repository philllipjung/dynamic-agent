# OpenTelemetry RUM 자동화 구현 가이드

## 1. CDN만으로 자동계측 (완전 무설정)

### 특징
- ✅ HTML에 스크립트 태그만 추가
- ✅ 소스 수정 불필요
- ✅ 빠른 구현
- ⚠️ localStorage 계측 안됨

### 자동 계측되는 항목
- 사용자 상호작용 (클릭, 입력, 폼 제출)
- 네트워크 요청 (Fetch, XMLHttpRequest)
- 페이지 로드 및 성능 메트릭
- 네비게이션

## 2. CDN 자동계측 + 소스 수정 최소화 + localStorage

### 특징
- ✅ HTML에 스크립트 추가
- ✅ localStorage까지 계측
- ⚠️ localStorage 래핑 코드 필요 (최소 수정)
- ⚠️ CDN만으로는 완전한 자동화 어려움

### 구현 코드

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>RUM 자동계측 + localStorage</title>

    <!-- OpenTelemetry Web SDK CDN -->
    <script src="https://cdn.jsdelivr.net/npm/@opentelemetry/sdk-web@0.45.1/dist/sdk-web.min.js"></script>

    <script>
        // 1. 기본 SDK 시작
        const sdk = new opentelemetry.sdk.WebSDK({
            resource: new opentelemetry.sdk.Resource({
                'service.name': 'rum-web-app',
                'service.version': '1.0.0'
            }),
            instrumentations: [],
            traceExporter: new opentelemetry.sdk.trace.ConsoleSpanExporter()
        });
        sdk.start();

        // 2. localStorage 자동 계측 (이 코드만 추가)
        (function() {
            const tracer = opentelemetry.trace.getTracer('localStorage-wrapper');

            // setItem 래핑
            const originalSetItem = localStorage.setItem;
            localStorage.setItem = function(key, value) {
                const span = tracer.startSpan('localStorage.setItem', {
                    attributes: {
                        'localStorage.key': key,
                        'localStorage.value.size': value.length,
                        'localStorage.operation': 'set'
                    }
                });
                try {
                    originalSetItem.call(this, key, value);
                    span.setStatus({ code: SpanStatusCode.OK });
                } catch (error) {
                    span.recordException(error);
                    span.setStatus({ code: SpanStatusCode.ERROR, message: error.message });
                } finally {
                    span.end();
                }
            };

            // getItem 래핑
            const originalGetItem = localStorage.getItem;
            localStorage.getItem = function(key) {
                const span = tracer.startSpan('localStorage.getItem', {
                    attributes: {
                        'localStorage.key': key,
                        'localStorage.operation': 'get'
                    }
                });
                try {
                    const value = originalGetItem.call(this, key);
                    span.setAttribute('localStorage.found', value !== null);
                    if (value) {
                        span.setAttribute('localStorage.value.size', value.length);
                    }
                    span.setStatus({ code: SpanStatusCode.OK });
                    return value;
                } catch (error) {
                    span.recordException(error);
                    span.setStatus({ code: SpanStatusCode.ERROR, message: error.message });
                } finally {
                    span.end();
                }
            };

            // removeItem 래핑
            const originalRemoveItem = localStorage.removeItem;
            localStorage.removeItem = function(key) {
                const span = tracer.startSpan('localStorage.removeItem', {
                    attributes: {
                        'localStorage.key': key,
                        'localStorage.operation': 'remove'
                    }
                });
                try {
                    originalRemoveItem.call(this, key);
                    span.setStatus({ code: SpanStatusCode.OK });
                } catch (error) {
                    span.recordException(error);
                    span.setStatus({ code: SpanStatusCode.ERROR, message: error.message });
                } finally {
                    span.end();
                }
            };

            // clear 래핑
            const originalClear = localStorage.clear;
            localStorage.clear = function() {
                const span = tracer.startSpan('localStorage.clear', {
                    attributes: {
                        'localStorage.operation': 'clear'
                    }
                });
                try {
                    originalClear.call(this);
                    span.setStatus({ code: SpanStatusCode.OK });
                } catch (error) {
                    span.recordException(error);
                    span.setStatus({ code: SpanStatusCode.ERROR, message: error.message });
                } finally {
                    span.end();
                }
            };

            console.log('✅ localStorage 자동 계측 활성화!');
        })();

        console.log('🚀 RUM 자동 계측 시작!');
    </script>
</head>
<body>
    <h1>RUM 자동계측 + localStorage 테스트</h1>

    <button onclick="localStorage.setItem('user', 'john'); alert('저장됨!')">
        localStorage 저장
    </button>
    <button onclick="alert(localStorage.getItem('user'))">
        localStorage 불러오기
    </button>
    <button onclick="localStorage.removeItem('user'); alert('삭제됨!')">
        localStorage 삭제
    </button>
</body>
</html>
```

### 추가된 localStorage 계측
- `setItem`: 키, 값 크기
- `getItem`: 키, 값 존재 여부, 값 크기
- `removeItem`: 키
- `clear`: 전체 삭제 이벤트

#### 2) src/index.js (RUM 자동 초기화)

```javascript
/**
 * OpenTelemetry RUM 완전 자동화 번들
 *
 * 사용법:
 * 1. npm run build로 번들링
 * 2. HTML에 <script src="dist/rum-bundle.js"></script> 추가
 * 3. 끝! 모든 것이 자동으로 계측됨
 */

import { WebSDK } from '@opentelemetry/sdk-web';
import { Resource } from '@opentelemetry/resources';
import { ConsoleSpanExporter, OTLPTraceExporter } from '@opentelemetry/exporter-trace-otlp-http';
import { SEMRESATTRS_SERVICE_NAME, SEMRESATTRS_SERVICE_VERSION } from '@opentelemetry/semantic-conventions';

// ========== 설정 (필요시 수정) ==========
const RUM_CONFIG = {
    serviceName: 'rum-web-app',
    serviceVersion: '1.0.0',
    environment: 'production',

    // Collector URL (필요시 수정)
    collectorUrl: 'http://localhost:4318/v1/traces',

    // Console 출력 여부 (개발: true, 프로덕션: false)
    enableConsole: true
};

// ========== localStorage 자동 계측 ==========
function setupLocalStorageInstrumentation(tracer) {
    const operations = ['setItem', 'getItem', 'removeItem', 'clear'];

    operations.forEach(operation => {
        const original = localStorage[operation];

        localStorage[operation] = function(...args) {
            const span = tracer.startSpan(`localStorage.${operation}`, {
                attributes: {
                    'localStorage.operation': operation,
                    'localStorage.key': args[0] || '',
                    [operation === 'setItem' ? 'localStorage.value.size' : 'exists']: args[1]?.length || false
                }
            });

            try {
                const result = original.apply(this, args);
                span.setStatus({ code: 0 }); // OK
                return result;
            } catch (error) {
                span.recordException(error);
                span.setStatus({ code: 1, message: error.message }); // ERROR
                throw error;
            } finally {
                span.end();
            }
        };
    });
}

// ========== sessionStorage 자동 계측 ==========
function setupSessionStorageInstrumentation(tracer) {
    const operations = ['setItem', 'getItem', 'removeItem', 'clear'];

    operations.forEach(operation => {
        const original = sessionStorage[operation];

        sessionStorage[operation] = function(...args) {
            const span = tracer.startSpan(`sessionStorage.${operation}`, {
                attributes: {
                    'sessionStorage.operation': operation,
                    'sessionStorage.key': args[0] || ''
                }
            });

            try {
                const result = original.apply(this, args);
                span.setStatus({ code: 0 });
                return result;
            } catch (error) {
                span.recordException(error);
                span.setStatus({ code: 1, message: error.message });
                throw error;
            } finally {
                span.end();
            }
        };
    });
}

// ========== SDK 초기화 ==========
function initRUM() {
    // Resource 생성
    const resource = new Resource({
        [SEMRESATTRS_SERVICE_NAME]: RUM_CONFIG.serviceName,
        [SEMRESATTRS_SERVICE_VERSION]: RUM_CONFIG.serviceVersion,
        'deployment.environment': RUM_CONFIG.environment
    });

    // Exporter 설정
    const exporters = [];
    if (RUM_CONFIG.enableConsole) {
        exporters.push(new ConsoleSpanExporter());
    }
    exporters.push(new OTLPTraceExporter({
        url: RUM_CONFIG.collectorUrl
    }));

    // SDK 생성 및 시작
    const sdk = new WebSDK({
        resource,
        instrumentations: [], // 자동 계측 활성화
        traceExporter: exporters.length === 1 ? exporters[0] : exporters
    });

    sdk.start();

    // localStorage/sessionStorage 계측 설정
    const tracer = sdk.getTracer('rum-auto-bundle');
    setupLocalStorageInstrumentation(tracer);
    setupSessionStorageInstrumentation(tracer);

    console.log('🚀 RUM 완전 자동 계측 시작!', {
        service: RUM_CONFIG.serviceName,
        version: RUM_CONFIG.serviceVersion,
        features: ['user-interactions', 'network', 'performance', 'localStorage', 'sessionStorage']
    });

    return sdk;
}

// ========== 자동 시작 ==========
// 스크립트 로드 시 자동으로 초기화
const rumSDK = initRUM();

// 글로벌하게 노출 (필요시 사용)
window.__RUM_SDK__ = rumSDK;
window.__RUM_CONFIG__ = RUM_CONFIG;

export default rumSDK;
```

## 3가지 방법 비교

| 항목 | 방법1: CDN만 | 방법2: CDN+localStorage | 방법3: 번들링 |
|------|-------------|----------------------|--------------|
| **구현 난이도** | ★☆☆☆☆ 매우 쉬움 | ★★☆☆☆ 쉬움 | ★★★☆☆ 보통 |

| **localStorage** | ❌ | ✅ | ✅ |
| **sessionStorage** | ❌ | ❌ | ✅ |
| **자동계측 범위** | 기본만 | 기본+localStorage | 완전자동 |
| **번들 크기** | 외부 로딩 | 외부 로딩 | ~500KB |
| **프로덕션** | ⚠️ 제한있음 | ⚠️ 제한있음 | ✅ 권장 |
| **유지보수** | 어려움 | 어려움 | 쉬움 |

## 4. Cloudflare Workers로 완전 무설치 자동계측

### 특징
- ✅ HTML 수정 불필요
- ✅ JavaScript 소스 수정 불필요
- ✅ 사용자가 아무것도 하지 않아도 됨
- ✅ 모든 HTML 응답에 자동으로 RUM 스크립트 주입
- ⚠️ DNS/프록시 설정 필요
- ⚠️ Cloudflare 계정 필요

### 아키텍처

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────┐
│ 사용자 브라우저 │────▶│ Cloudflare Worker │────▶│ 원본 서버    │
└─────────────┘     │  (RUM 자동 주입)  │     │  (그대로)    │
                    └──────────────────┘
                            │
                            ▼
                    ┌──────────────┐
                    │ Collector    │
                    │ (트레이스 수신)│
                    └──────────────┘
```

### 구현

#### 1) Cloudflare Worker 스크립트

```javascript
// worker.js

/**
 * RUM 자동 계측 Cloudflare Worker
 * 모든 HTML 응답에 RUM 스크립트를 자동으로 주입합니다.
 */

// RUM 스크립트 템플릿
const RUM_INJECT_SCRIPT = `
<script src="https://cdn.jsdelivr.net/npm/@opentelemetry/sdk-web@0.45.1/dist/sdk-web.min.js"></script>
<script>
(function() {
    const sdk = new opentelemetry.sdk.WebSDK({
        resource: new opentelemetry.sdk.Resource({
            'service.name': window.location.hostname,
            'service.version': '1.0.0',
            'rum.deployment': 'cloudflare-worker'
        }),
        instrumentations: [],
        traceExporter: new opentelemetry.sdk.trace.OTLPTraceExporter({
            url: 'https://your-collector.com:4318/v1/traces'
        })
    });

    sdk.start();

    // localStorage 자동 계측
    const tracer = sdk.getTracer('rum-auto');
    ['localStorage', 'sessionStorage'].forEach(storageType => {
        const storage = window[storageType];
        ['setItem', 'getItem', 'removeItem', 'clear'].forEach(op => {
            const original = storage[op];
            storage[op] = function(...args) {
                const span = tracer.startSpan(\`\${storageType}.\${op}\`, {
                    attributes: { 'storage.key': args[0] || '' }
                });
                try {
                    const result = original.apply(this, args);
                    span.end();
                    return result;
                } catch (error) {
                    span.recordException(error);
                    span.end();
                    throw error;
                }
            };
        });
    });

    // 네트워크 자동 계측 (SDK가 자동으로 처리)
    console.log('✅ RUM 자동 계측 시작 (Cloudflare Worker)');
})();
</script>
`;

// HTML인지 확인하는 함수
function isHTML(headers) {
    const contentType = headers.get('content-type') || '';
    return contentType.includes('text/html');
}

// </head> 태그 찾기
function findHeadEnd(html) {
    const headEndIndex = html.indexOf('</head>');
    return headEndIndex !== -1 ? headEndIndex : null;
}

// HTML에 RUM 스크립트 주입
function injectRUMScript(html) {
    const headEndIndex = findHeadEnd(html);

    if (headEndIndex === null) {
        // </head>가 없으면 <body> 앞에 주입
        const bodyStartIndex = html.indexOf('<body');
        if (bodyStartIndex !== -1) {
            return html.slice(0, bodyStartIndex) +
                   RUM_INJECT_SCRIPT +
                   html.slice(bodyStartIndex);
        }
        // 둘 다 없으면 </title> 뒤에 주입
        const titleEndIndex = html.indexOf('</title>');
        if (titleEndIndex !== -1) {
            return html.slice(0, titleEndIndex + 8) +
                   RUM_INJECT_SCRIPT +
                   html.slice(titleEndIndex + 8);
        }
        // 그래도 없으면 그냥 반환
        return html;
    }

    // </head> 앞에 주입
    return html.slice(0, headEndIndex) +
           RUM_INJECT_SCRIPT +
           html.slice(headEndIndex);
}

// Cloudflare Worker 메인 핸들러
export default {
    async fetch(request, env, ctx) {
        // 원본 서버로 요청 전달
        const response = await fetch(request);

        // HTML 응답만 처리
        if (!isHTML(response.headers)) {
            return response;
        }

        // HTML 내용 가져오기
        const originalHTML = await response.text();

        // RUM 스크립트 주입
        const instrumentedHTML = injectRUMScript(originalHTML);

        // 수정된 HTML 반환
        return new Response(instrumentedHTML, {
            status: response.status,
            statusText: response.statusText,
            headers: response.headers
        });
    }
};
```

### 장점

1. **완전 무설치**: 사용자가 아무것도 할 필요 없음
2. **중앙 관리**: 모든 설정이 CloudFlare Worker에서 관리
3. **즉시 적용**: Worker만 수정하면 모든 사이트에 반영
4. **CDN 성능**: CloudFlare의 전 세계 CDN 활용
5. **SSL/TLS**: 무료 SSL 자동 제공
6. **DDoS 방어**: CloudFlare의 보안 기능 활용

### 단점

1. **CloudFlare 의존**: CloudFlare를 반드시 사용해야 함
2. **DNS 변경**: 도메인 DNS를 CloudFlare로 변경해야 함
4. **비용**: 무료 플랜은 100,000 requests/day
5. **복잡성**: 초기 설정이 다소 복잡

### 사용 시나리오

#### 1. RUM SaaS 서비스
```
- 고객이 도메인만 알려주면 자동으로 계측 시작
- 고객이 HTML을 수정할 필요 없음
- 서비스 제공자가 Worker만 관리
```

```
- Worker에서 로직만 수정하면 즉시 적용
- 각 사이트마다 수정 불필요
```

## 주의사항

1. **CORS 문제**: Collector URL이 다른 도메인이면 CORS 설정 필요
2. **샘플링**: 프로덕션에서는 샘플링률 조정 권장 (기본 100%)
3. **민감 정보**: 개인정보가 포함된 데이터는 마스킹 처리 필요
4. **성능**: 번들링 시 code splitting으로 로딩 속도 최적화
5. **CloudFlare Workers**: 무료 플랜 한도(10만 requests/day) 확인 필요
6. **DNS 변경**: CloudFlare Workers 사용 시 DNS 반드시 변경 필요
