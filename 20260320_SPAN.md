# Span Create Feature (스팬 생성 기능)

## 개요

런타임에 실행 중인 JVM의 메서드에 OpenTelemetry Span을 동적으로 생성하는 기능입니다. ByteBuddy를 사용하여 재시작 없이 실시간으로 계측(instrumentation)할 수 있습니다.

**버전**: 1.1.0
**생성일**: 2026-03-20
**수정일**: 2026-03-22

## 구성 요소

### 1. ByteBuddyController

REST API를 제공합니다.

**위치**: `agent-server/src/main/java/com/javaagent/server/controller/ByteBuddyController.java`

```java
@PostMapping("/createSpan")
public Map<String, Object> createSpan(@RequestBody CreateSpanRequest request) {
    String className = request.getClassName();
    String methodName = request.getMethodName();

    // ByteBuddyAgent 호출
    String result = ByteBuddyAgent.createSpan(null, className, methodName);

    return Map.of(
        "success", true,
        "message", result
    );
}
```

### 2. ByteBuddyAgent

ByteBuddy를 사용하여 런타임에 바이트코드를 수정합니다.

**위치**: `bytebuddy-agent/src/main/java/com/javaagent/bytebuddy/ByteBuddyAgent.java`

```java
public static String createSpan(String pid, String className, String methodName) {
    String methodKey = className + "." + methodName;

    if (appliedMethods.contains(methodKey)) {
        return "SUCCESS: Span already created for " + methodKey;
    }

    // AgentBuilder로 Advice 적용
    new AgentBuilder.Default()
        .with(AgentBuilder.RedefinitionStrategy.RETRANSFORMATION)
        .type(ElementMatchers.named(className))
        .transform(new AgentBuilder.Transformer() {
            @Override
            public DynamicType.Builder<?> transform(
                    DynamicType.Builder<?> builder,
                    TypeDescription typeDescription,
                    ClassLoader classLoader,
                    JavaModule module,
                    ProtectionDomain protectionDomain) {

                // Helper 주입
                if (classLoader != null) {
                    injectHelper(classLoader);
                }

                // SpanAdvice 적용
                return builder.visit(
                    Advice.to(com.javaagent.bytebuddy.advices.SpanAdvice.class)
                        .on(ElementMatchers.named(methodName)
                                .and(ElementMatchers.isMethod())
                                .and(ElementMatchers.not(ElementMatchers.isStatic())))
                );
            }
        })
        .installOn(instrumentation);

    appliedMethods.add(methodKey);
    return "SUCCESS: Span created for " + methodKey;
}
```

### 3. SpanAdvice

메서드 실행 전후에 스팬 생성/종료 코드를 삽입합니다.

**위치**: `bytebuddy-advice/src/main/java/com/javaagent/bytebuddy/advices/SpanAdvice.java`

```java
public class SpanAdvice {

    @Advice.OnMethodEnter
    public static void onMethodEnter(
            @Advice.Origin Method method,
            @Advice.This Object target
    ) {
        try {
            String className = target.getClass().getName();
            String methodName = method != null ? method.getName() : "unknown";
            String spanName = className + "." + methodName;

            SpanHelper helper = new SpanHelper();
            helper.startSpan(spanName);

        } catch (Exception e) {
            System.err.println("[SpanAdvice] Error: " + e.getMessage());
        }
    }

    @Advice.OnMethodExit(onThrowable = Throwable.class)
    public static void onMethodExit(@Advice.Throwable Throwable throwable) {
        try {
            SpanHelper helper = new SpanHelper();
            if (throwable != null) {
                helper.recordException(throwable);
            }
            helper.endSpan();
        } catch (Exception e) {
            System.err.println("[SpanAdvice] Exit error: " + e.getMessage());
        }
    }
}
```

### 4. SpanHelper

런타임에 OpenTelemetry API를 호출하여 스팬을 생성합니다.

**위치**: `bytebuddy-advice/src/main/java/com/javaagent/bytebuddy/helper/SpanHelper.java`

```java
public class SpanHelper {
    private static Tracer tracer;
    private Span span;
    private Scope scope;

    public SpanHelper() {
        if (tracer == null) {
            // OpenTelemetry Tracer 초기화
            tracer = OpenTelemetry.getInstance()
                .getTracerProvider()
                .get("java-agent", "1.0.0");
        }
    }

    public void startSpan(String spanName) {
        span = tracer.spanBuilder(spanName)
            .setParent(Context.current())
            .startSpan();
        scope = span.makeCurrent();
    }

    public void endSpan() {
        if (scope != null) {
            scope.close();
            scope = null;
        }
        if (span != null) {
            span.end();
            span = null;
        }
    }

    public void recordException(Throwable throwable) {
        if (span != null) {
            span.recordException(throwable);
            span.setStatus(StatusCode.ERROR, throwable.getMessage());
        }
    }
}
```

## Helper 클래스 주입

ByteBuddyAgent는 타겟 ClassLoader에 Helper 클래스를 주입합니다:

```java
private static boolean injectHelper(ClassLoader targetLoader) {
    try {
        String[] helperClasses = {
            "com.javaagent.bytebuddy.helper.SpanAttributeHelper",
            "com.javaagent.bytebuddy.helper.SpanHelper",
            "com.javaagent.bytebuddy.helper.SpanLinkHelper"
        };

        for (String helperClassName : helperClasses) {
            byte[] classBytes = loadHelperClassBytes(helperClassName);
            injectClassIntoLoader(targetLoader, helperClassName, classBytes);
        }

        return true;
    } catch (Exception e) {
        System.err.println("[ByteBuddy] Failed to inject helpers: " + e.getMessage());
        return false;
    }
}

private static void injectClassIntoLoader(ClassLoader targetLoader,
                                         String className,
                                         byte[] classBytes) throws Exception {
    Method defineClass = ClassLoader.class.getDeclaredMethod(
        "defineClass", String.class, byte[].class, int.class, int.class
    );
    defineClass.setAccessible(true);
    defineClass.invoke(targetLoader, className, classBytes, 0, classBytes.length);
}
```

## 제한사항

### 1. 정적 메서드

현재 정적 메서드는 지원하지 않습니다:

```java
// 정적 메서드는 계측되지 않음
public static void staticMethod() { }
```

### 2. 중복 계측

이미 계측된 메서드는 다시 계측되지 않습니다:

```java
if (appliedMethods.contains(methodKey)) {
    return "SUCCESS: Span already created for " + methodKey;
}
```

### 3. ClassLoader 격리

Helper 클래스는 타겟 ClassLoader로 주입되어야 합니다.
